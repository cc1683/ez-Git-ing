What is Git?

Git is a version control system for tracking changes in computer files and coordinating work on those files among multiple people. --Wikipedia

Installation

For Linux user

sudo apt-get install git-all

For Mac and Window user

Can directly download from here

Configure Git

Set the git username and email (globally)

git config -g user.name "username"
git config -g user.email user@gmail.com

if dont want to set globally just remove the -g in the command

Check your git username and email

git config -l

Basic Git commands

Initiate an empty git repository

git init

Check the working tree status

git status

Check branch

git branch
